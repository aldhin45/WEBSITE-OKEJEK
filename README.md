Oke-Jek adalah aplikasi transportasi online berbasis web yang menghubungkan pengguna dengan mitra pengemudi secara realtime. Proyek ini dibangun untuk mensimulasikan proses pemesanan, manajemen driver, dan kalkulasi tarif dinamis.

* **User Panel:** Pemesanan perjalanan (Motor/Mobil/Barang), estimasi harga otomatis, riwayat pesanan.
